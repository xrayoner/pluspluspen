# ++PEN v1.0.0 Güncelleştirme Raporu

## Özet
- Açılış toolbar görünümü kullanıcı referansı `Adsız.png` baz alınarak yeniden hizalandı.
- Referans görselden kırpılan toolbar arka planı uygulama içinde doğrudan kullanılmaya başlandı.
- Kalem, silgi, geri al, yinele, kaydet ve renk seçimi için tıklanabilir hit alanları korundu.
- Sürüm bilgileri, update feed ve paketleme scriptleri `1.0.0` sürümüne güncellendi.

## Dağıtım
- Uygulama sürümü: `1.0.0`
- Update paketi adı: `pluspluspen_update_1.0.0.zip`
- Setup adı: `++PENSetup_1.0.0.exe`
